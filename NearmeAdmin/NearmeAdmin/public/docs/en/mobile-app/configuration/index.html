Configuration

Global configuration

Open the app.config.ts file located insrc/app and change the value of the following variables:

  • SERVER_URL - It must match the declared value in Dockerfile.

  • APP_ID - It must match the declared value in Dockerfile.

  • GOOGLE_MAPS_API_KEY - You can use the same key generated during the admin portal configuration or create a new one from here.

  • BANNER_ID - Optional. Leave it empty if you do not need to show Google advertising. Otherwise, go to AdMob portal, create a new app and create a new banner ID.

  • TRACKING_ID - Google Analytics tracking ID. Optional. Leave it empty if you do not want to measure usage statistics.

  • HEADER_COLOR - Color for the status bar (Android only). Optional. Any hexadecimal value is valid.

  • DEFAULT_UNIT - Default distance unit. Possible values: km, mi.

  • DEFAULT_LANG -Default language Possible values: en,es. If you want to add another language, create a new file in the src/assets/i18n folder. Then copy and paste the contents of the en.json file into the new file and edit the texts accordingly.

Update metadata

Before publishing it is necessary to update the config.xml. Change the package ID and name as shown below:

<widget id="com.yourdomain.appname" version="1.0.0">
    <name>Your app name</name>
</widget>

Setup push notifications

Open config.xml and update the value of ParseServerUrl and ParseGcmSenderId.

Example:

<!-- Both values must match the Dockerfile -->
<preference name="ParseServerUrl" value="https://nearme.quanlabs.com/parse/" />
<preference name="ParseGcmSenderId" value="XXXXXXXXXXX" />

It is important that ParseServerUrl has the / character at the end, so that the notifications work correctly.

Setup Facebook Login

In config.xml edit APP_NAME and APP_ID as shown below:

<plugin name="cordova-plugin-facebook4" spec="^2.1.0">
    <variable name="APP_ID" value="YOUR_APP_ID" />
    <variable name="APP_NAME" value="YOUR_APP_NAME" />
    <variable name="ANDROID_SDK_VERSION" value="4.+" />
</plugin>

For more information on how to generate these values visit the Ionic Native documentation.

Change colors

Changing colors is as easy as editing the color map $colors in variables.scss file.

$colors: (
    primary:    #e91e63,
    secondary:  #f4f4f4,
    danger:     #f53d3d,
    light:      #f4f4f4,
    accent:     #ccc,
    dark:       #7e7e7e,
    darker:       #444,
    fb:         #3b5998,
);

To update the toolbar color edit the variables $gradient-bg and $secondary-bg in app.scss.

Update Icon and SplashScreen

Replace the files located in the /resources folder and in src/assets/img.

For more information about how to manage the resources please visit the Ionic Documentation.
