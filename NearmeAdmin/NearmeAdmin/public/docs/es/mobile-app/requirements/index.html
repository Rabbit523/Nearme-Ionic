Requerimientos

Ionic depende de Node.js para funcionar correctamente, por lo tanto empezaremos visitando la página oficial de Node.js y descargaremos el binario correspondiente a nuestro sistema operativo.

Teniendo instalado Node.js, procedemos a instalar Ionic con el siguiente comando:

npm install -g ionic

Esperamos a que termine la instalación. Una vez finalizado, usando la terminal de línea de comandos navegamos hacia la carpeta donde se encuentra el código fuente de la aplicación móvil y ejecutamos lo siguiente:

ionic serve

Este comando creará un compilado funcional para entornos web. Espera unos segundos y deberás ver la aplicación corriendo en tu navegador web.

Si quieres consultar más detalles la guía oficial de Ionic cuenta con toda la información necesaria.
